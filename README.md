# Tip Calculator (Python)

A simple command-line Python project that calculates how much each person should pay when splitting a bill, including tip.

## 🚀 Features
- Takes total bill amount as input
- Allows user to choose tip percentage
- Splits the bill among multiple people
- Calculates exact amount per person

## 🧠 What I Learned
- Type conversion (int/float)
- Taking user input in Python
- Mathematical operations
- Writing basic logic for real-world problems

## ⚙️ How It Works
1. User enters the total bill amount
2. User selects the tip percentage
3. User enters number of people
4. Program calculates and displays how much each person should pay

## ▶️ How to Run
1. Clone the repository
2. Run the Python file:

```bash
